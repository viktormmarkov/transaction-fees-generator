### Prerequisite
1. node@8.11.2
2. npm@5.6.0

#### Development environment
##### Visual Studio Code
You will need the following add-ons on Visual Studio Code:
- Beautify
- ESLint


Clone the repository

Install all dependencies

```
cd transaction-fees-generator
npm install
```

Run the application
```
node app.js input.json
```

Run the tests
```
npm run test
```
